# Next.js Practice Projects

Welcome to my Next.js practice repository! This repository contains a collection of small projects and exercises to help me master Next.js, a powerful React framework for building modern web applications.

## Projects Overview

Here’s a list of the projects included in this repository:

1. **Add Fonts**
   - **Description**: Practice working with variable fonts and Google Fonts in Next.js.
   - **Folder**: `projects/Add-Fonts_Variable-Fonts_Google-Fonts/`

2. **Dynamic Routes**
   - **Description**: Implement dynamic routes, nested dynamic routes, and catch-all routes.
   - **Folder**: `projects/Dynamic_Routes-Nested_Dynamic_Routes-Catch_All_Routes/`

3. **Event Handling**
   - **Description**: Learn how to handle events, create functions, and manage state in Next.js.
   - **Folder**: `projects/Event_Handling-Creating_Functions-Managing_States/`

4. **Image Handling**
   - **Description**: Practice managing and optimizing images in Next.js.
   - **Folder**: `projects/Manage_Images/`

5. **Layouts**
   - **Description**: Master layouts and nested layouts in Next.js.
   - **Folder**: `projects/Mastering_Layout-Nested_Layouts/`

6. **Routing**
   - **Description**: Explore routing with pages, nested routes, and grouping pages.
   - **Folder**: `projects/Routing_with_Pages-Nested_Routes-Grouping_Pages/`

7. **Server and Client Components**
   - **Description**: Understand the difference between server and client components in Next.js.
   - **Folder**: `projects/Server_Components-Client_Components/`

8. **My First Next.js App**
   - **Description**: A simple Next.js app to get started with the framework.
   - **Folder**: `projects/my-first-next-app/`

## How to Run These Projects

1. **Clone the Repository**
   ```bash
   git clone https://github.com/RavinduChamodWickramage/practice-next-js.git
   cd practice-next-js
   ```

2. **Navigate to a Project**
   ```bash
   cd projects/<project-folder-name>
   ```

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Run the Development Server**
   ```bash
   npm run dev
   ```

5. **Open in Browser**
   Visit http://localhost:3000 in your browser to view the project.
